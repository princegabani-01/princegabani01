# Digital Agency Portfolio Website

A modern and responsive digital agency website built using HTML, CSS, JavaScript, and Bootstrap.

## 🌟 Project Overview

This project is a professional Digital Agency and Portfolio Website designed to showcase services, portfolio projects, team members, testimonials, and contact information. The website features a clean and modern UI with responsive design for desktop, tablet, and mobile devices.

## ✨ Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action Buttons
* About Section
* Services Section
* Portfolio Showcase
* Team Members Section
* Testimonials Slider
* Contact Form
* Modern UI Design
* Mobile Responsive Layout
* Smooth Animations

## 🛠 Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Bootstrap Icons
* AOS Animation Library
